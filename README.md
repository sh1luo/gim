## 这是什么？

这是一个 IM （即时通讯）程序*DEMO版本*的后台应用程序，具备基本的消息收发，用户上、下线通知功能，易扩展。

可以配合前端DEMO使用，移步 [前端地址](https://github.com/sh1luo/im_front)

## 我应该如何构建？

```
git clone https://github.com/sh1luo/im_backend.git
cd im_backend
go run main.go
```

> 1.克隆项目到本地。
>
> 2.进入项目目录。
>
> 3运行 main.go 主程序
